### 安装依赖

python.exe -m pip install --upgrade pip

[opencv-python](https://pypi.org/project/opencv-python/4.8.0.74/)

pip install opencv-python==4.8.0.74

pip install -i https://pypi.tuna.tsinghua.edu.cn/simple pynput
